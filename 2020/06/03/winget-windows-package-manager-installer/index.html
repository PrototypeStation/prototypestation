TUTORIAL – Winget Windows Package Manager installer

Posted on June 3, 2020 by prototypestation

Su linux esisteva apt-get per installare pacchetti come Steam. Ora Windows ha winget che è molto simile all’installatore di pacchetti di Linux. Basterà digitare sul nuovo prompt dei comandi denominato Terminal (giusto per essere un po più Linuxiani) la parola winget e ci verranno forniti i vari comandi di winget se questo è installato. Facciamo un esempio: dobbiamo cercare il nome del pacchetto di una applicazione che vogliamo installare su Windows quindi dall’app Terminal digitiamo winget search <nome pacchetto>. Premendo invio ci verranno forniti tutti i pacchetti inerenti a <nome pacchetto>. Successivamente per installarlo dovremo digitare winget install <nome pacchetto visualizzato dalla ricerca>. In automatico il Terminal scaricherà il file .exe e lo installerà.
